# Movie-serach-app

네이버 영화 API를 활용한 영화 검색 서비스입니다.
<br>
<br>
axios로 http 요청을 하였으며 cors error를 해결하기 위해

- localhost:3000 : 클라이언트
- localhoast:3001 : express 서버

위와 같이 클라이언트와 서버를 분리 하였으며 서버를 통해 데이터를 호출하였습니다.

## 기능

키워드 검색 시 네이버 영화 API의 응답을 받아 리스트를 노출합니다.

## 사용기술

- React
- express
- axios
- styled-components
- react-icons
